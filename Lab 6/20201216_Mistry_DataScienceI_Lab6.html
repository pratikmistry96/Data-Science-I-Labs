2) We will first do a PCA analysis. Scale, perform PCA, and plot the variance explained and the cumulative variance explained by the PCs. How many PCs are produced in the analysis? Approximately how many PCs are required to explain 90% of the variance in the data?¶

In [20]:
## Scale nci60 data
scaler = StandardScaler() # initiate scaling function
pca = PCA() # Initiate PCA

scaler.fit(nci60_data) # Fit the scaling 
nci60_data_scale = scaler.fit_transform(nci60_data) # Transform the data

pca.fit(nci60_data_scale) # Fit pca to transformed data
pca_exp_var = pca.explained_variance_ratio_ # Extract explained variance ratio
pca_cum_var = np.cumsum(pca_exp_var) # Take the cumulative sum

plt.figure(figsize=(8,6)) # Set figure size
plt.plot(np.arange(1,1+len(pca_exp_var)), pca_cum_var) # Plot pc number and cumulative explained variance
plt.xlabel('PC Number') # Label x
plt.ylabel('Cumulative Explained Variance') # label y
plt.title('Cumulative Explained Variance by PC Number') # Add title
plt.show() # Show plot

Answer:¶

From the plot of cumulative explained variance by PC number, it looks like approximately 40-45 PCs are required to explain 90% of the variance in the data
